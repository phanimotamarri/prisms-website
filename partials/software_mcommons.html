Materials Commons Open Source Software


Materials Commons is our data repository and collaboration platform. In developing Materials Commons we created a suite of libraries and platforms for the platform. Many of these software libraries are platform agnostic.


Here at the PRISMS Center we love open source software (OSS). As a way of giving back to the community we have open sourced the entire Materials Commons platform and support tools. Here you will find libraries and tools ranging from an Erlang based monitoring solution to a general purpose configuration library designed for Go. We put a lot of effort into making sure our libraries are usable by the wider community.


If you are interested in learning more about Materials Commons, our software tools, or our general OSS initiative drop us a line at our contacts page. We'd love to hear from you!


Below you find a list of all the software and modules we have open sourced for Materials Commons. You can also go to our page at GitHub and poke around our extensive set of repositories.


Materials Commons
This is the main repository for the Materials Commons web application.


mcstore
mcstore handles file uploads and downloads. It deals with data integrity, ensuring that your uploads proceed quickly, and file de-duplication. mcstore handles extensions to the Flow file upload protocol.


flow.js
This is a fork of the wonderful flowjs file uploader. Here we added in Materials Commons specific needs, as well as extending it to compute block hashes when uploading files.


mcfs
mcfs is our original file uploader. There are still many great ideas contained in the original, and we refer to it often when adding extensions to mcstore.


gen_stomp
Do you think Erlang rocks! Would you like to use Erlang with stomp but in a way that looks like an Erlang Gen Server? gen_stomp is our implementation of a Gen Server for STOMP. Wondering what STOMP is? Head on over to Stomp and check out this interesting protocol.


AngularStomp
An AngularJS module for interacting with WEB-STOMP.


gohandy
gohandy is set of, ahem, handy utilities for Go. You can find modules that make it easier to deal with the file system, encryption, and other nuggets.


materials
materials is our command line interface to Materials Commons. It's currently going through some changes as we update it for our coming general release.


tika-dm3-parser
The Tika DM3 Parser library is an extension for Tika for extracting tags and meta data from a DM3 image.


send_file
send_file is a Erlang client and server for sending files across the network. Not currently being used by our site, its still an interesting utility for handling file transfers. In particular you can see how to do block handling in secure sockets under Erlang.


resource_discovery
resource_discovery is an Erlang based distributed resource discovery server.


process-monitor
process-monitor is a small Erlang based process monitoring tool. It allows you to set up groups of services to monitor, and provides restarts and monitoring using Erlangs built in supervisors. You can can set up separate supervisors for different groups of modules.


handyman
handyman is a set of utilities and modules for Erlang. It includes nifs for getting environment variables from Windows and Linux, translating between strings and Erlang terms, and other 'handy' utilities.
