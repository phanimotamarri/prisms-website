Fatigue Behavior

The Fatigue Behavior Use Case group is investigating and developing capabilities for predicting the role of alloying and microstructure on the low cycle fatigue response and the growth of microstructurally small fatigue cracks in Mg alloys.


Integration Approach

The above figure provides a schematic of the PRISMS fatigue behavior scientific workflow integration protocol. As indicated in the figure, outputs from CASM calculations are used as inputs to ParaDis and PRISMS-VMM. The ParaDis calculations will be used as inputs to the PRISMS-CPFE and PRISMS-Plasticity Codes. Key experimental measurements have also been identified for inputs to PRISMS-CPFE and VMM. This integration protocol is very similar to that described for the Tensile Behavior use case and the integration linkages described in the tensile use case section identify the same linkages. In addition for prediction of fatigue behavior, inputs to PRISMS-VMM will be determined by fitting fatigue crack growth "Paris Law" constants obtained from PRISMS-CPFE to critical experiments.

Computational Tools

  • PRISMS KS-DFT: Real Space Kohn-Sham Density Functional theory code
  • PRISMS PF: Phase field methods code
  • PRISMS Plasticity: Continuum Plasticity and Crystal Plasticity code
